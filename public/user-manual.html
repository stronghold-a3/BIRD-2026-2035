What is BIRD 2026–2035?
Platform overview and purpose
▾

BIRD 2026–2035 is a strategic planning platform built for the Bangsamoro Investment Roadmap Development — the 10-year investment blueprint for the Bangsamoro Autonomous Region in Muslim Mindanao (BARMM). The platform integrates:

📊 Real-Time KPIs
MEL Dashboard with live monitoring across BSC perspectives
✦ AI-Powered Analysis
SWOT generation, strategy formulation, and insights
🔄 Systems Thinking
CLD builder, Meadows leverage points, archetypes
🎯 Balanced Scorecard
4-perspective tracking with calibrated targets
🏗 PAPs Management
Programs, activities, projects with budget tracking
🤖 BIRD AI
Expert AI consultant for BARMM investment strategy

MEL Dashboard
Monitoring, Evaluation, and Learning — your real-time command center
📊
Default Landing View
The MEL Dashboard is the default view when you enter any strategic plan workspace. It provides "Total Clarity" — a unified monitoring, evaluation, and learning command center for your entire strategic plan.
📊
Dashboard Panels
Five integrated monitoring panels
▾
Panel A — Pareto Vital Few KPIs
Displays 6 headline KPIs derived from the BIRD BSC. Ring progress charts show current vs. target. Status badges: On Track (green), Watch (blue), Behind (amber), Critical (red). Tap/click any KPI for details.
Panel B — BSC Leverage Points
Shows critical leverage points across 4 BSC perspectives (Financial, Stakeholder, Internal Process, Learning & Growth). Each row displays current vs. 2030 and 2035 targets.
Panel C — Priority Action Plan
Lists the 10 priority actions for 2026 (Foundation Phase). Filter by Priority (Critical / High / Medium), Status (In Progress / Development / Not Started), and Quarter (Q1–Q4).
Panel D — Feedback Loop Health Monitor
Displays active Causal Loops (R1, R2, B1, B2…) from the Systems Thinking layer. Shows activation percentage and health status.
Panel E — Phase Progress Tracker
Timeline view of all 3 BIRD implementation phases with budget allocations.
Phase 1
Foundation Building
2026–2028 · ₱35–45B
Phase 2
Acceleration
2029–2032 · ₱50–65B
Phase 3
Consolidation
2033–2035 · ₱35–50B

📐
Resilience Index (RI) Scoring
BIRD's official mathematical methodology
▾
SWOT FactorFormulaMeaning
StrengthsRI = (Impact × Likelihood) / 5Resilience capacity (1–5)
OpportunitiesRI = √(Impact × Likelihood)Resilience via opportunity exploitation (1–5)
WeaknessesRisk Score = Impact × LikelihoodInternal risk exposure (1–25)
ThreatsVI = (Impact² × Likelihood) / 25External vulnerability (1–5)
ℹ️
Auto-Color Coding
Items are automatically color-coded by score for quick prioritization. Critical items (highest scores) appear at the top of the Impact view.

Systems Thinking
Causal Loop Diagrams, Archetypes, and Leverage Points
💡
Key Insight
Every strategic problem is embedded in a system. The Systems Thinking module helps you move beyond listing problems to understanding the causal structures that generate them — so your interventions address root causes, not just symptoms.
🔗
Building a Causal Loop Diagram
Add nodes, create links, detect loops
▾
1
Add Nodes
Click "+ Add Node" (or double-click the canvas). Enter the variable name (e.g., "Investment Approvals", "BHB Certification Quality"). Drag nodes to position them.
2
Add Links
Hover over a node until you see the connector point. Drag to another node to create a causal link. Set polarity: + (reinforcing) or – (counteracting).
3
Detect Loops
The system automatically detects Reinforcing (R) and Balancing (B) loops. Loops are highlighted with directional arrows.

Generating Strategies

Adding Objectives & KPIs
Step-by-step scorecard building
▾

Adding Objectives

  1. Navigate to Balanced Scorecard
  2. Click + Add Objective in any perspective panel
  3. Fill in: Objective name, Description, Weight (1.0–2.0)
  4. Link to a strategic option (from Strategy Matrix) if applicable

Adding KPIs to an Objective

  1. Expand an objective by clicking it
  2. Click + Add KPI
  3. Fill in: KPI name, Unit, Baseline value, 2030 target, 2035 target
  4. Set frequency (monthly / quarterly / annually)
✦
AI Suggest KPIs
Expand any objective and click "AI Suggest KPIs" — BIRD AI analyzes the objective text and suggests relevant, measurable indicators tailored to BARMM's investment context.

Troubleshooting
Common issues and solutions
Platform won't load
▾
  1. Clear browser cache (Ctrl+Shift+Del / Cmd+Shift+Delete)
  2. Try a different browser
  3. Check your internet connection
  4. Disable browser extensions (especially ad blockers)
Can't sign in
▾
  1. Check that your email is verified (check inbox for verification link)
  2. Try "Forgot password?" to reset
  3. Try "Send magic link" for password-free access
  4. Contact BOI-MTIT if your account is locked
AI Assistant not responding
▾
  1. Ensure you have an active internet connection
  2. Try refreshing the page
  3. If the issue persists, the AI service may be temporarily unavailable — try again in a few minutes
Plans not saving / sync issues
▾
  1. Check the sync status indicator in the topbar (should be green)
  2. If offline, your changes are saved locally and will sync when you reconnect
  3. Click Force Sync in Settings → Data Management
Data export fails
▾
  1. Ensure you have selected at least one section for export
  2. Check that your plan has content (SWOT, objectives, PAPs)
  3. Try a different export format
Error: "Session expired"
▾
Your login session has timed out — click Sign In to re-authenticate. Your data is preserved locally.
